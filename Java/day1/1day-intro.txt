 Java 특징 
 : 플랫폼 독립적  , JVM(Java Virtual Machine) 이 OS와 통역 역할
    -> 크로스 플랫폼이라고도 한다 
	-> "Write Once, Run Anywhere" 
   객체지향언어 
   멀티스레드(Multi Thread)를 지원 
   Garbage Collection을 지원 
 ------------------------------
 Java Compile :   javac HelloWorld.java 
				-> HelloWorld.class 생성 ( jvm이 읽을 파일 )
			
Java  Runtime :  java HelloWorld 
				-> 실행결과

- 객체지향 ( Object-Oriented ) 
  : 시스템을 분석,설계하는 기법 

    객체를 기본 단위로 상정하고
    객체와 객체의 관계를 중심으로 
    시스템을 분석 설계하는 기법을 객체지향이라고 한다 
참고) UML(unified modeling language) -> 
		객체 지향 분석 설계시 사용하는 모델링 언어
